Tags semânticas

        Não possuem uma pré-definição específica,
        sendo seu uso apenas para demonstrar
        o significado de determinado bloco html.

    - < nav > para barras de navegação
    - < header > para cabeçalho
    - < main > para conteúdo principal
    - < aside > para conteúdo a parte/auxiliar 
    - < article > para divisões de conteúdo
    - < section > para seções de uma pagina    
    - < footer > para rodapé
    - < figure > para imagens
